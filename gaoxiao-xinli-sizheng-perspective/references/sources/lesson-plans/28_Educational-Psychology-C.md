#### 一、课程简介（包括面向对象、开设目的、主要内容、课程特色等内容，500字以内）
《教育心理学》是面向汉语国际教育专业二年级本科生开设的一门专业选修课，它为学生提供作为教师职业所需的教育心理学基础知识的同时，也为学生学习教育类课程提供必要的知识准备。其目的在于：通过对教育心理学基础和知识的学习，了解教育心理学的对象、性质及研究方法；明确学习教育心理学的意义，认识心理现象发生、发展以及相互作用的规律；初步培养学生运用教育心理学的基本原理分析和解决在教育教学过程中遇到的各种心理问题的能力；帮助其树立科学的心理发展观和教育观，为将来的从教生涯提供心理科学方面的观念指导。

本课程主要涉及绪论、学生心理、学习理论、学习心理、教学心理和教师心理。本课程紧密结合汉语国际教育专业的培养目标，具有鲜明的基性、应用性与发展性。以奠定教师职业所需的核心理学知识为出发点，系统涵盖从绪论、学生与学习心理到教学与教师心理的完整内容体系。课程不仅致力于帮助学生掌握教育心理学的对象、性质、研究方法及心理现象规律，更核心聚焦于培养其运

二、课程挖掘的思政资源分析（2000字以内）

教育心理学是研究教育教学情境中教与学的基木心规律的科学，其课程内容蕴含丰富的思政元素。围绕“立德树人”人才培养的根本标准，课程思政资源主要体现在以下五个方面：

一、以唯物辩证法为指导，塑造科学教育观

辩证看待环境与教育的作用：课程中关于个体差异、智力发展等内容，深刻阐释了遗传、环境与教育三者之间的关系。通过创新关键期研究等，引导学生深刻理解教育在塑造人、改变人、促进人全面发展中的决定性力量，树立“教育兴则国家兴，教育强则国家强”的信念。

理解内因与外因的辩证统一：在学习动机、知识建构等理论中，强调学生主体性（内因）与教师引导、社会文化环境（外因）的互动关系。引导学生认识到教育不是单向灌输，而是在尊重学生主体性基础上，通过创设良好环境（外因），激发其内在动力（内因），实现教学相长。

实践是认识的源泉与动力：综合行为主义学习理论、班杜拉的社会学习理论等，强调实践在知识获得、技能形成和态度改变中的核心地位。引导学生理解“纸上得来终觉浅，绝知此事要躬行”，倡导理论联系实际的教学方法，呼应国家倡导的实践育人导向。

二、弘扬中华优秀传统教育思想，坚定文化自信

# 教学内容

## 1. 设计思路
以学生为中心，讲授与互动研讨相结合。以游戏导入激发兴趣，引出智力结构模式及定义，通过案例引发学生对单一评价标准的批判性思考。教学核心在于系统梳理理论脉络，对比早期测量观与当代系统观，重点聚焦多元智能理论在国际中文教育的应用，并通过“六艺”的古今对话彰显中华教育智慧。随后，以张桂梅、北斗等本土化案例为依托，引导学生运用三元智力理论和PASS模型分析教育实践中的智力协同与认知加工过程，强化理论迁移能力。最终总结升华，引导学生理解经典、立足创新，践行“有教无类”理念。

## 2. 教学重点
本课的教学重点是多元智能理论的内涵与应用；以及个体智力差异对教学设计的启示。

## 3. 教学难点
本节课的教学难点是区分心理测量理论与信息加工理论的本质差异；以及理解三元智力理论中三种亚理论的互动关系。

## 4. 对重点、难点的处理
在突破多元智能理论重点时，通过国际中文教育案例与“六艺”的跨时空对照，具象化理论内涵，并引导学生设计差异化数学活动，实现从理论到教学启示的转化。针对理论辨析难点，采用案例研讨论法，对比分析心理测量理论与信息加工理论的本质差异。

四、自行补充的内容（可选）

专业知识与思政元素的有机融合

1. 倡导教育公平与尊重个体差异  
强调每个人都有独特的智能组合和发展路径，教育者应“多一把尺子，多一批人才”，为不同潜能的学生提供发展平台，体现“有教无类”的公平理念。

2. 促进文化理解与认同  
将西方现代心理学理论（多元智能）与中国传统文化（六艺）进行创造性联系，既说明了理论的普适性，也凸显了中国文化元素的独特价值（如“御”的综合智能、“虚心有节”的品格）。

3. 培养家国情怀与社会责任感  
钱学森、袁隆平是爱国、创新、求实、奉献、协同、育人的科学家精神的杰出代表，他们的成就展现了高水平的智力如何服务于国家重大需求；张桂梅成长的例子，生动诠释了教育工作者的崇高使命感和无私奉献精神，其成功是智力与崇高师德完美结合的典范。

4. 培养战略思维与规划能力  
PASS模型的“计划系统”和《孙子兵法》的“谋定而后动”，以及北斗系统的建设过程，都强调了目标设定、策略规划和动态调整能力的重要性，这对学生个人成长和国家长远发展都至关重要。

5. 提升批判性思维能力  
通过展示智力理论从简单到复杂、从单一到多元的演进过程，以及对其局限性的分析（如早期测验的简化），鼓励学生不盲从权威，理解理论的相对性和发展性，培养独立思考能力。
