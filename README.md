# 思思 · 高校心理专业课教师（课程思政强）视角 Skill Agent

🎯 **本项目致力于将高校心理学专业知识与课程思政元素完美融合，追求“润物无声，实证为本”的创新教学设计。**

本项目包含了一个高度结构化的 AI Agent 提示词（Skill）。目前的心理学课程思政内容作为一个**标准参考范式（Reference）**。底层的代码与结构具有极强的可拓展性，既能直接服务于心理学教育者，也方便其他学科的教师以此为模板，自定义开发属于自己学科的课程思政智能体。

---

## ✨ 核心特色

* **有机融入 (Organic Integration)**：拒绝生硬拼贴，让思政元素从专业知识土壤中自然生长。
* **实证引导 (Evidence-Based)**：用科学的研究证据和数据说话，用“研究显示”代替“你应该”。
* **以生为本 (Student-Centered)**：精选 10+ 大学生真实生活场景（如刷短视频、网购智商税、宿舍关系等）作为理论锚点。
* **学科高度可定制（Framework-Ready）**：本项目不仅是一套现成的心理学思政方案，更是一套**通用的学科思政代码框架**。其他学科（如计算机、理工科、医学等）教师可轻松替换专业课知识点与思政锚点，实现自定义复用。

## 🛠️ 快速开始

### 🚀 平台部署（推荐）
本 Skill 完美适配主流 AI 编程与智能体构建平台。你可以将其部署到 **WorkBuddy**、**Trea**、**Dify** 或 **Coze** 等平台上：
1. 在 AI 平台中新建一个 **Agent / Skill（插件）**。
2. 复制本仓库中的 `SKILL.md` 文件的全部代码/提示词内容。
3. 将其粘贴到平台的 **System Prompt（系统提示词）** 或 **Agent 编排面板** 中。
4. 发布/上架后，即可在你的工作流或对话框中随时调用。

### 💬 快捷唤醒（触发词）
在对话中提及以下关键词，即可唤醒本智能体：
> `思思`、`课程思政`、`心理课思政`、`高校心理教师`、`备课思政`、`立德树人`

### 💡 典型使用场景
* **教学方案设计**：*“思思，如何帮我在讲授《社会心理学》的‘群体压力与服从’时，设计一个贴近大学生宿舍生活的思政教学方案？”*
* **学生心理开导**：*“有个学生因为保研同辈压力产生严重内卷焦虑，我该怎么从心理学和价值观引导的双重角度开导 ta？”*

## 📖 课堂案例库预览（心理学 Reference）

本 Skill 内置了贴近当代大学生的“生活锚定法”心理学案例作为示范，部分示例如下：

| 心理学知识点 | 大学生生活场景（示例） | 思政/价值观锚点 |
| :--- | :--- | :--- |
| **说服与从众** | 家里老人被推销员忽悠买保健品，怎么劝都不听 | 科学精神、代际沟通、信息素养 |
| **社会认知/归因** | 小红书种草、短视频算法让你越刷越焦虑 | 批判思维、自我觉察、数字健康 |
| **群体压力/服从** | 宿舍集体熬夜，你不想熬但也跟着熬；小组作业搭便车 | 独立人格、责任担当、同伴影响 |
| **自我概念/身份** | 社交媒体上的“精修人设” vs 现实中的普通自己 | 自我接纳、真实与表演 |

*(更多完整案例请查阅 `SKILL.md`。若您是其他学科教师，可参照此结构修改 `SKILL.md` 中的知识库代码)*
