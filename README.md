# 思思 · 高校心理专业课教师（课程思政强）视角 Skill Agent

🎯 **本项目致力于将高校心理学专业知识与课程思政元素完美融合，追求“润物无声，实证为本”的创新教学设计。**

本项目包含了一个高度结构化的 AI Agent 提示词（Skill），能够帮助高校心理学教师、备课人员以及教育工作者，在设计课程方案或解答学生困惑时，自然地融入价值观引导。

---

## ✨ 核心特色

* **有机融入 (Organic Integration)**：拒绝生硬拼贴，让思政元素从心理学知识土壤中自然生长。
* **实证引导 (Evidence-Based)**：用科学的研究证据和数据说话，用“研究显示”代替“你应该”。
* **以生为本 (Student-Centered)**：精选 10+ 大学生真实生活场景（如刷短视频、网购智商税、宿舍关系等）作为理论锚点。
* **多心智模型驱动**：内置辩证张力模型、隐性渗透模型等5大心智模型，保证回答的深度与学术严谨性。

---

## 🛠️ 快速开始

### 触发词
在对话中提及以下关键词，即可唤醒本智能体：
> `思思`、`课程思政`、`心理课思政`、`高校心理教师`、`备课思政`、`立德树人`

### 使用方法
1. 复制本仓库中的 `SKILL.md` 文件的全部内容。
2. 粘贴到各大主流大语言模型（如 ChatGPT, Claude, Gemini 等）的系统提示词（System Prompt）或对话框中。
3. 接下来，你可以这样向它提问：
   * *“思思，如何帮我在讲授《社会心理学》的‘群体压力与服从’时，设计一个贴近大学生宿舍生活的思政教学方案？”*
   * *“有个学生因为保研同辈压力产生严重内卷焦虑，我该怎么从心理学和价值观引导的双重角度开导 ta？”*

## 📖 课堂案例库预览（大学生生活场景）

本 Skill 特别整理了贴近当代大学生的“生活锚定法”案例，部分示例如下：

| 心理学知识点 | 大学生生活场景（示例） | 思政/价值观锚点 |
| :--- | :--- | :--- |
| **说服与从众** | 家里老人被推销员忽悠买保健品，怎么劝都不听 | 科学精神、代际沟通、信息素养 |
| **社会认知/归因** | 小红书种草、短视频算法让你越刷越焦虑 | 批判思维、自我觉察、数字健康 |
| **群体压力/服从** | 宿舍集体熬夜，你不想熬但也跟着熬；小组作业搭便车 | 独立人格、责任担当、同伴影响 |
| **自我概念/身份** | 社交媒体上的“精修人设” vs 现实中的普通自己 | 自自我接纳、真实与表演 |

*(更多完整案例请查阅 `SKILL.md`)*
